A plugin desgined to be a replacement for the default WordPress gallery in a responsive fashion.  This provides a slideshow for the [gallery] shortcode and on the attachment pages along with the ability to add extra slides based on the post tags or category.

== Installation ==

= Manual Installation =

1. Upload the entire `/elastigallery` directory to the `/wp-content/plugins/` directory.
2. Activate ElastiGallery through the 'Plugins' menu in WordPress.
